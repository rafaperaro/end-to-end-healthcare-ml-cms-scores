# End-to-End Healthcare ML: CMS Scores

## Overview
This was a group oroject for which we applied an end-to-end machine learning workflow to analyze
healthcare CMS quality scores. The goal was to uncover patterns in provider
performance and translate data insights into meaningful business
implications.

## Workflow
- Market and domain research
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning modeling
- Unsupervised learning for pattern discovery
- Interpretation of results

## Skills Demonstrated
- Applied machine learning
- Data cleaning and preprocessing
- Model evaluation and interpretation
- Business-oriented analytical thinking

## Tools
- Python
- Pandas
- NumPy
- scikit-learn
- Matplotlib
- Seaborn
